# 🦉 LearnLock - Ingliz tili o'rganish ilovasi

## Ilova haqida
LearnLock - bolalar uchun inglizcha so'zlarni o'rganish ilovasi.
- 🃏 **Kartalar** - 15 ta so'zni ko'rish va yodlash
- 🧠 **Test** - 10 ta savol bilan bilimni sinash

## APK qilish uchun ko'rsatmalar

### Variant 1: Codemagic (tavsiya etiladi)
1. github.com da yangi repo yarating
2. Bu papkani zip qilib yuklang
3. codemagic.io ga kiring → GitHub bilan login
4. Loyihani tanlang → Build → APK tayyor!

### Variant 2: FlutLab.io
1. flutlab.io ga kiring
2. "Import from ZIP" → bu papkani zip qiling va yuklang
3. "Build APK" tugmasini bosing

### Variant 3: Flutter o'rnatilgan bo'lsa
```
flutter pub get
flutter build apk --release
```
APK: build/app/outputs/flutter-apk/app-release.apk
